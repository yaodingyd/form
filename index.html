Form Validation and Building by JSON

A Demo page to show form validation and building on the fly using JSON send by backend.

Build

Building takes object like following specify form attributes like action, and fields inside form.

        
{
  "action": "index.html",
  "method": "post",
  "fieldsets": [{
    "name": "About You",
    "id": "about",
    "fields": [
    {
      "type": "text",
      "name": "firstName",
      "id": "firstName",
      "caption": "First Name",
      "placeholder": "John",
      "class": "first-name"
    }]
  }]
}

Validate

Validate takes an array of to-be-validated items and initialize rules and add event listeners to elements.

The object must contains an id(DOM id to append event listeners), rules which can be pipelined. You can include customed message for each rule, if not there are default messages.

        
[{
  "id": "firstName",
  "rules": "required|name|maxLength[35]",
  "messages": {
    "required": "First Name is required",
    "name": "Invalid First Name."
  }
}]

var formValidation = new validate('myform', validation[0]);

The default classname for error message is error-message. Additionally you can add a hint, that would display when the input field is focused, and the default class name is hint.

Update form

Both validate and build can be updated with new data.Validate would check each cached field and add/remove based whether it's on DOM. Build has a naive Virtual DOM implementation that would only change what is different in each fieldset, thus preserving the exsiting inputs.
